## Overview

This project presents an interactive Excel dashboard designed to deliver clear, consistent, and actionable insights on regional and store-level sales performance. It tracks monthly revenue trends, year-over-year changes, store comparisons, and top product gains and losses while maintaining scalability for expanding data sources.

## Features

* **Interactive Filters:** Region, store, and time-period slicers for dynamic analysis.
* **KPI Tracking:** Revenue, monthly trends, YoY performance, and product-level insights.
* **Store Performance Comparison:** Visual breakdown of high and low performers.
* **Product Analysis:** Identifies top-selling products and those driving losses.
* **Clean, Scalable Structure:** Data prepared and modeled to support future growth.

## Workflow

1. **Define Dashboard Purpose:** Aligned visuals with leadership needs.
2. **Select Key Metrics:** Focused KPIs to simplify decision-making.
3. **Plan for Growth:** Created a robust data model for expanding datasets.
4. **Prepare Data:** Cleaned and transformed using Excel tools (Power Query, PivotTables).
5. **Build Visuals:** Designed primary charts and supporting visuals.
6. **Dashboard Design:** Final layout emphasizes clarity, interactivity, and storytelling.

## Tools Used

* Microsoft Excel

  * Power Query
  * Pivot Tables
  * Slicers & Timelines
  * Excel Charts

This project demonstrates end-to-end business intelligence skills, including data preparation, KPI definition, dashboard design, and visual storytelling in Excel.
